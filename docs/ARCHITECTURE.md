# 多智能体协作G代码生成系统架构

## 系统概述

本系统采用多智能体协作架构，基于 LangGraph 实现状态机和流程控制，专注于机加工G代码的智能生成。

## 架构设计

### 1. 智能体角色分工

#### 🏭 ProcessPlannerAgent (工艺规划专家)
- **职责**: 分析零件特征，制定加工工艺路线
- **能力**:
  - 识别加工特征（孔、槽、轮廓、曲面等）
  - 制定加工顺序
  - 选择加工方法（粗加工、半精加工、精加工）
  - 估算加工时间
- **输入**: 零件描述、材料信息、精度要求
- **输出**: 工艺规划方案（JSON格式）

#### 🔧 ToolExpertAgent (刀具专家)
- **职责**: 选择合适的刀具和切削参数
- **能力**:
  - 刀具类型推荐（铣刀、钻头、丝锥等）
  - 切削参数优化（转速、进给、切深）
  - 刀具路径策略选择
  - 刀具寿命预估
- **输入**: 工艺规划方案
- **输出**: 刀具清单和切削参数表

#### 🛤️ PathPlannerAgent (路径规划师)
- **职责**: 生成优化的刀具路径
- **能力**:
  - 生成粗加工路径（Zig-Zag、螺旋等）
  - 生成精加工路径（轮廓、等高线等）
  - 生成钻孔路径
  - 路径优化（空行程最短）
- **输入**: 刀具清单、零件几何信息
- **输出**: 刀具路径数据（坐标点序列）

#### 📝 GCodeGeneratorAgent (G代码生成器)
- **职责**: 将路径数据转换为标准G代码
- **能力**:
  - 支持主流数控系统（FANUC、SIEMENS、MAZAK等）
  - 生成完整的NC程序
  - 优化代码结构
  - 添加必要的辅助功能
- **输入**: 刀具路径数据
- **输出**: 标准G代码程序

#### ✅ SafetyValidatorAgent (安全验证器)
- **职责**: 验证G代码的安全性和合理性
- **能力**:
  - 碰撞检测
  - 刀具干涉检查
  - 参数范围验证
  - 程序逻辑检查
- **输入**: G代码程序
- **输出**: 验证报告（通过/警告/错误）

### 3. 状态管理

系统使用统一的状态对象在智能体之间传递信息：

```python
class MachiningState(TypedDict):
    # 输入信息
    part_description: str          # 零件描述
    material: str                  # 材料信息
    precision_requirements: dict   # 精度要求
    
    # 工艺规划
    process_plan: Optional[dict]   # 工艺方案
    features: Optional[list]       # 特征列表
    
    # 刀具信息
    tool_list: Optional[list]      # 刀具清单
    cutting_parameters: Optional[dict]  # 切削参数
    
    # 路径数据
    tool_paths: Optional[dict]     # 刀具路径数据
    
    # G代码
    gcode_program: Optional[str]   # G代码程序
    
    # 验证
    validation_result: Optional[dict]  # 验证结果
    
    # 元数据
    current_agent: str             # 当前智能体
    iteration: int                 # 迭代次数
    messages: list                 # 消息历史
```

### 4. 技术栈

- **框架**: LangGraph 1.0 (状态机编排)
- **LLM**: 豆包大模型 (工艺知识推理)
- **工具**: 自定义专业工具集
- **存储**: 短期记忆 + 可选持久化

### 5. 特性

✅ 模块化设计 - 每个智能体独立可测试  
✅ 专业性强 - 领域知识深度集成  
✅ 可扩展 - 易于添加新功能  
✅ 安全可靠 - 多层验证机制  
✅ 开源友好 - 清晰的代码结构  

## 使用示例

```python
# 输入零件描述
input_data = {
    "part_description": "铝合金零件，包含一个直径20mm深30mm的孔和一个50x30mm的槽",
    "material": "6061-T6铝合金",
    "precision_requirements": {
        "hole_tolerance": "H7",
        "surface_finish": "Ra1.6"
    }
}

# 系统自动执行多智能体协作
result = await graph.ainvoke(input_data)

# 输出G代码和验证报告
print(result["gcode_program"])
print(result["validation_result"])
```
