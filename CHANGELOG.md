# 更新日志

本项目的所有重要更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2025-03-XX

### 新增

#### 核心功能
- 多智能体协作架构（5个专业智能体）
- 自然语言特征识别
- 智能刀具选择与参数优化
- 刀具路径自动规划
- FANUC 标准G代码生成
- 多层安全验证机制

#### 支持的加工特征
- 孔特征：通孔、盲孔
- 槽特征：矩形槽
- 型腔特征：矩形型腔
- 轮廓特征：外轮廓

#### 支持的材料
- 铝合金（6061-T6、7075等）
- 碳钢（45#、1045等）
- 不锈钢（304、316等）

#### 技术实现
- 基于 LangGraph 1.0 的状态机编排
- 16个专业工具函数
- PostgreSQL 记忆持久化
- HTTP API 接口
- 流式响应支持（SSE）

### 性能指标
- 特征识别准确率：85%+
- 安全验证覆盖率：100%
- 简单任务响应时间：2-3秒
- 复杂任务响应时间：5-8秒

### 文档
- 架构设计文档
- 性能验证报告
- 部署指南
- API 参考
- 使用示例

## 版本说明

### 版本号格式

- **主版本号**：不兼容的 API 更改
- **次版本号**：向后兼容的功能新增
- **修订号**：向后兼容的问题修复

### 更新类型

- **新增 (Added)**：新功能
- **更改 (Changed)**：对现有功能的变更
- **弃用 (Deprecated)**：即将移除的功能
- **移除 (Removed)**：已移除的功能
- **修复 (Fixed)**：bug 修复
- **安全 (Security)**：安全相关修复
